# 项目经历

## BPM 业务流程管理平台

**项目描述：**
专注于企业复杂业务流程的可视化定义、自动化任务分配与跨部门协作审批。平台包含流程设计器、任务审批中心、数据看板等核心模块，支持通过拖拽配置实现采购、请假等各类业务流程的线上化与标准化。

**技术栈：**
Vue3 + TypeScript + Ant Design Vue + Axios + Vite + Vue Router + AntV X6 + Pinia + Prisma

**项目工作：**

- 基于 Vue3 + TypeScript 搭建模块化工程，通过 Vue Router 实现路由懒加载与动态权限拦截，集成 Pinia 状态管理替代 Vuex，提升类型安全性与代码可维护性。

- 基于 AntV X6 实现可视化流程设计器，支持流程节点的拖拽配置与连线编辑。结合动态表单引擎，实现条件网关、审批路径计算等复杂业务逻辑，支持表单字段的条件渲染与关联数据校验，实现业务流程的高度定制化。

- 设计并实现审批引擎，支持顺序审批、全员审批、任意审批三种审批模式，通过条件表达式实现流程分支的智能路由，满足不同业务场景的审批需求。

- 基于 TypeScript 对 Axios 进行二次封装，集成请求拦截器实现 Token 自动注入、统一错误处理及接口重试机制。通过类型约束接口返回结构，在编译期确定接口契约，减少业务层防御性代码，提升代码健壮性。

- 设计主题化方案（暗黑/亮色模式），基于 Ant Design Config Provider 与 localStorage 持久化配置，支持动态切换与全局响应式更新，提供良好的用户体验。

- 针对审批记录、流程日志等大数据量场景，基于 Vue3 Composition API 封装 useVirtualList Hook，通过动态计算滚动视窗与元素位置关系，实现仅渲染可见区域的虚拟列表组件，大幅提升页面性能。

- 负责平台数据可视化看板建设，基于 Ant Design Charts 实现流程状态、审批耗时与趋势等核心指标分析，支持管理侧快速洞察流程效率与瓶颈。

- 使用 Vite 构建工具优化项目构建速度，通过代码分割、懒加载等手段优化首屏加载性能，提升用户体验。
